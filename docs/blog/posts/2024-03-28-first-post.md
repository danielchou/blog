## 為什麼選擇 MkDocs？

MkDocs 有許多優點：

1. Markdown 支援
2. 簡潔的設計
3. 強大的擴充性
4. 完整的搜尋功能
5. 支援程式碼高亮

## 設置步驟

1. 建立 GitHub 倉庫
2. 設定 MkDocs
3. 自訂主題
4. 設定自動部署
