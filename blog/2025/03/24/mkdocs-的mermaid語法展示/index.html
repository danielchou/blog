    mermaid語法展示流程圖¶

    本文將展示Mermaid語法的使用方法。

    Diagrams¶

    流程圖 (left - right)：¶

    graph LR
    A[開始] --> B{判斷}
    B -->|Yes| C[執行]
    B -->|No| D[跳過]
    C --> E[結束]
    D --> E

    流程圖 (top - down)：¶

    flowchart TD
    A[Christmas] -->|Get money| B(Go shopping)
    B --> C{Let me think}
    C -->|One| D[Laptop]
    C -->|Two| E[iPhone]
    C -->|Three| F[fa:fa-car Car]

    sequenceDiagram

    使用者->>系統: 提供驗證
    系統-->>使用者: 登入成功

    使用 Mermaid 繪製活動圖：¶

    activityDiagram
    title 簡單活動流程
    start
    :讀取資料;
    if (資料有效?) then (yes)
      :處理資料;
      :儲存結果;
    else (no)
      :記錄錯誤;
    endif
    stop

    使用 Mermaid 繪製類別圖：¶

    classDiagram
    title 動物類別範例
    class Animal {
      +String name
      +Int age
      +eat()
      +sleep()
    }
    class Dog {
      +bark()
    }
    class Cat {
      +meow()
    }
    Animal <|-- Dog
    Animal <|-- Cat
